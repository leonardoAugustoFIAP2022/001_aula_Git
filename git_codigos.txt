GIT      = Ferrementa para Versionamento
GIT BASH = Ferramenta para o GIT
GIT HUB  = Rede Social onde posso enviar meus projetos do GIT

--------------------------------------------------------------------------
Registrar NOME e E-MAIL
--------------------------------------------------------------------------
git config --global user.email "Leonardo Augusto"
git config --global user.email "rm93485@fiap.com.br"

--------------------------------------------------------------------------
Consultar NOME e E-MAIL
--------------------------------------------------------------------------
git config user.name
git config user.email

--------------------------------------------------------------------------
Inicia o versionamento
--------------------------------------------------------------------------
git init

--------------------------------------------------------------------------
Adiciona "TODOS" os arquivos
--------------------------------------------------------------------------
git add .

--------------------------------------------------------------------------
Adiciona arquivos específicos
--------------------------------------------------------------------------
git add git_codigos.txt
(Caso haja outros arquivos digitar o nome do arquivos e separar com espaço)

--------------------------------------------------------------------------
Realiza o commit
--------------------------------------------------------------------------
git commit -m "primeiro commit git_codigos.txt"

--------------------------------------------------------------------------
Adiciona arquivos específicos
--------------------------------------------------------------------------
git add *.png

--------------------------------------------------------------------------
Só adiciona os arquivos que sofreram modificação
--------------------------------------------------------------------------
git add *
